Tipos de etiquetas en HTML y sus usos

¿Qué son las etiquetas de HTML?
Una etiqueta contiene mucha información. Le indica al navegador dónde empieza y dónde termina cada elemento, mientras que un atributo describe las características. Los elementos HTML suelen dividirse en tres partes. Esta combinación de las tres crea un elemento HTML.

Algunos tipos de etiquetas

  • < body>para el contenido.
  • < head>para información sobre el documento.
  • < br>para saltos de línea.
  • < H1>…< H6> para títulos dentro del contenido.
  • < img> para añadir imágenes al documento.
  • < ol> para listas ordenadas.
  • < ul> para listas desordenadas.
  • < li> para elementos dentro de la lista.
  • < p> para parágrafos.
  • < span> para estilos de una parte del texto
